Technologies involved in this project

Applications

Web server:

  • Web server: NodeJS & ExpressJS, Socket.IO for websockets
  • Reactive programming: RSVP promises & RxJs
  • Unit tests with Mocha

Front:

  • AngularJS &UI router
  • BootstrapCSS & Less CSS
  • Math.js & Function plot

Data analysis:

  • Python server
  • Scikit-learn
The data-analysis and web applications interact with the ZeroMQ distributed messaging library, which has Python and NodeJS bindings.

Architecture

Source code:

The project source is hosted at github.com/lucsorel/watts-app along other projects, have a look at them!

Containers:

The datamining and web servers are isolated in Docker containers. Their images are hosted on the lucsorel/watts-app Docker hub repository.

Hosting:

This application runs on an Amazon Web Service EC2 node (a small cheap one...), driven by the Docker Cloud cluster manager.
